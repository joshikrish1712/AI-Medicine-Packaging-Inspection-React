# AI-Medicine-Packaging-Inspection

## Project Overview
This project is a web-based application created to inspect medicine packaging and identify common issues such as spelling mistakes on labels, poor print quality, damaged packaging, broken seals, or irregularities in pills like unusual color, size, or texture.

The application is designed to simulate how technology can assist in quality checking during medicine packaging.

## Features
- Visual inspection of medicine packaging
- Checks for packaging defects and inconsistencies
- Simple and clean web interface
- Easy to extend for future improvements

## Technologies Used
- Frontend: React.js
- Backend: Node.js
- Styling: Tailwind CSS
- Build Tool: Vite

## How to Run the Project
1. Download or clone the repository
2. Install dependencies:
   npm install
3. Start the application:
   npm run dev
